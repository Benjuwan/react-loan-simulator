## ⚠ レビュー前に必ず読むこと：過去の誤指摘パターン

以下は**正しい挙動であるにもかかわらず、誤ってバグと判定されやすいパターン**です。
これらを「バグ」と報告することは厳禁です。必ず下記の理由を理解した上で判断してください。

### 誤指摘パターン1：年次集計で利息が一時的に増加する現象

**現象**: 年次集計グラフで、ある年の利息合計が前年より増加している

**誤った判断**: 「元利均等返済なら利息は単調減少するはずなのでバグ」

**正しい理解**:
- 金利変動が年の途中（例：11ヶ月目）に発生した場合、
  1年目 = 安い金利×N ヶ月 + 高い金利×(12-N) ヶ月
  2年目 = 高い金利×12ヶ月
  という非対称な集計になるため、2年目の利息合計が1年目を上回ることは数学的に正しい。
- 「単調減少」は月次・同一金利継続の場合の性質であり、金利変動×年次集計には適用されない。

**判断基準**:
  → 変動後の最初の「完全な1年」以降で利息が減少していれば正常。
  → 変動が起きた年をまたぐ1〜2年のみの増加は正常。

---

### 誤指摘パターン2：相対変化率を無視した「ほぼ横ばい」判定

**現象**: 金利変動幅が絶対値で小さく見える（例: 0.68% → 0.84%）

**誤った判断**: 「0.16%の変化はほぼ横ばいなので影響は軽微なはず」

**正しい理解**:
- 相対変化率は (0.84 - 0.68) / 0.68 ≈ 23.5% であり、利息計算への影響は大きい。
- 特に低金利帯（0%台）では絶対値差より相対倍率で考えること。

**判断基準**:
  → 金利変動の影響評価は必ず相対変化率（%）で行うこと。
  → 絶対値差（bps）だけを根拠に「誤差範囲」と判断しないこと。

---

### 誤指摘パターン3：5年ルール+125%ルールによる期末残債

**現象**: fullyRepaid が false、または期末に少額の元金・未払い利息が残る

**誤った判断**: 「35年後に残債があるのは計算バグ」

**正しい理解**:
- 125%ルールで返済額増加を上限で抑えた場合、返済が追いつかず期末に残債が生じることがある。
- これはルールの仕様上の帰結であり、正常な挙動。
- 実際の金融機関でも最終月に一括清算が求められるケースがある。

**判断基準**:
  → 残債額が小さく（元金の1%未満程度）かつ125%ルール適用シナリオなら正常。
  → 残債が元金の数%を超える場合は計算ロジックを要確認。

---

### 誤指摘パターン4：充当順序に関する誤解

**現象**: 未払い利息発生中に「元金充当額が0」になっている月がある

**誤った判断**: 「元金が減っていないのはバグ」

**正しい理解**:
- 返済額 < 当月利息 の場合、元金への充当はゼロが正しい。
- 充当順序は「①当月利息 → ②未払い利息残高の返済 → ③元金」が正しい仕様。

## リファレンス実装の仕様（比較基準）

以下の仕様を基準として、レビュー対象コードと比較してください。

### 月利計算
```
monthlyRate = annualRate / 100 / 12
```

### 月次返済額（元利均等）
```
payment = principal × monthlyRate × (1 + monthlyRate)^n / ((1 + monthlyRate)^n - 1)
※ monthlyRate = 0 の場合: payment = principal / n
```

### 5年ルール
- 返済額の見直しは借入開始から60ヶ月ごと（61ヶ月目、121ヶ月目、181ヶ月目…）
- 見直し時の計算ベース: 「その時点の元金残高」「残返済月数」「現行金利」
- 見直し前は返済額固定（金利が変わっても返済額は変わらない）

### 125%ルール
- 見直し後の新返済額 = min(本来の返済額, 前回返済額 × 1.25)
- 下限はなし（金利低下時は制限なく減額される）

### 未払い利息
- 当月利息 = 元金残高 × monthlyRate（未払い利息残高には利息不加算）
- 返済額 ≥ 当月利息 の場合:
    余剰 = 返済額 - 当月利息
    未払い利息残高への充当 = min(余剰, 未払い利息残高)
    元金充当 = 余剰 - 未払い利息充当
- 返済額 < 当月利息 の場合:
    元金充当 = 0
    新規未払い利息 = 当月利息 - 返済額
    未払い利息残高 += 新規未払い利息

## 検証シナリオ

以下の4シナリオで、リファレンス実装の出力値とレビュー対象コードの出力値を比較してください。

### シナリオA：金利固定・基本確認
```
借入額: 3,000万円 / 期間: 35年 / 金利: 全期間 1.0% 固定
5年ルール: あり / 125%ルール: あり

確認項目:
- 初月返済額（リファレンス値: 84,685円）
- 全期間を通じて返済額が変化しないこと
- 利息が月次で単調減少すること
- 元金残高が月次で単調減少すること
- 未払い利息が一度も発生しないこと
```

### シナリオB：年途中の金利変動（誤指摘パターン1・2の検証）
```
借入額: 1,810万円 / 期間: 30年
金利: 0.68%（1〜10ヶ月目） → 0.84%（11ヶ月目〜）
5年ルール: あり / 125%ルール: あり

確認項目:
- 1年目の年次利息合計
- 2年目の年次利息合計（1年目より増加することを確認）
- 3年目以降は単調減少に戻ることを確認
- 未払い利息が発生しないこと（0.84%程度では発生しない）
```

### シナリオC：未払い利息の発生（コアロジック検証）
```
借入額: 4,000万円 / 期間: 35年
金利: 0.5% → 24ヶ月目に 3.0% → 60ヶ月目に 3.5%
5年ルール: あり / 125%ルール: あり

確認項目:
- 未払い利息が初めて発生する月（24〜60ヶ月の間のいずれか）
- 60ヶ月目時点の未払い利息累積額
- 61ヶ月目（5年ルール見直し後）の返済額が前回の1.25倍以内であること
- 未払い利息が発生している月の元金充当が0であること
```

### シナリオD：ペアローン合算検証
```
Aさん: 借入額2,500万円 / 期間35年 / 0.5%→24ヶ月目に2.0%
Bさん: 借入額2,000万円 / 期間30年 / 0.5%→24ヶ月目に2.0%
5年ルール: あり / 125%ルール: あり（両者共通）

確認項目:
- 初月合算返済額 = Aさん初月返済額 + Bさん初月返済額
- 合算残高が各人の残高の単純合計と一致すること
- Bさんの完済後（361ヶ月目以降）、合算値がAさんのみの値と一致すること
```

**差異#N**
- シナリオ: X

### 3. 総合判定

「ロジック一致」/ 「軽微な差異あり（許容範囲）」/ 「重大な差異あり（要修正）」

重大な差異の判断基準:
- 月次返済額の誤差が1円を超える
- 未払い利息の発生有無の判定が異なる
- 5年ルール見直し月のタイミングがずれている
- 125%上限の適用可否が異なる

## 注意事項

- 円未満の端数処理（切り捨て・四捨五入・切り上げ）はシステムにより異なる場合があります。
  1円未満の差異は「端数処理の違い」として許容範囲とし、バグ扱いしないこと。
- 「直感的におかしい」と感じた場合は、まず「誤指摘パターン」の該当可否を確認してから報告すること。
- シナリオBの「利息一時増加」は特に誤指摘されやすいため、慎重に判断すること。
