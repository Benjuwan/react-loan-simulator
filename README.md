# 住宅ローン シミュレーター (変動金利・ペアローン対応)

## 制作経緯
昨今の円安や世界情勢に伴う住宅ローンの金利上昇傾向を受け、金利変動が「支払い総額」や「月々の支払内訳」、そして「未払利息」の発生にどのような影響を与えるかを直感的に視覚化するために開発されました。
特に、変動金利特有の「5年ルール」や「125%ルール」が引き起こすリスクを可視化することに重点を置いています。

## 設計のポイント（計算ロジック）
- **変動金利シナリオの表現**: `LoanConditions.scenarios` に `monthOffset` と `interestRate` を定義し、指定月以降の適用金利を切り替えます。
- **元利均等返済の月次計算**: `calculateLoan()` は元利均等返済の支払額・利息・元金・残高を月次で計算し、支払期間中の変動を反映します。
- **5年ルール**: 初回60ヶ月経過後（61ヶ月目）に返済額を見直します。固定支払額モードが有効な場合は再計算せず、指定額を維持します。
- **125%ルール**: 返済額見直し時、新しい支払額は直前の返済額の1.25倍を上限とします。
- **未払利息の管理**: 当月の利息が月額返済額を超える場合、差額を未払利息として蓄積。以後の返済では未払利息を元金より優先して返済します。
- **夫婦それぞれの独立計算と合算表示**: `calculateLoan()` を夫・妻それぞれに実行し、`mergeMonthlyDetails()` で月ごとに合算。合算表示と個別表示を切り替えて比較できます。

## 類似・競合ツール一覧
銀行系と比較・情報サイト系の2カテゴリに分けています。

### 銀行系（自行商品への申込誘導が主目的）
#### [三菱UFJ銀行](https://www.bk.mufg.jp/kariru/jutaku/sim/sim.html)
- 月額返済額・総返済額・借入可能額の試算
- 5年ルール・125%ルールの説明ページあり（シミュレーターへの実装はなし）

#### [みずほ銀行](https://www.mizuhobank.co.jp/loan_housing/net_simulation/index.html)
- 月額返済額・残高推移表（年齢別）

#### [三井住友銀行](https://www.smbc.co.jp/kojin/jutaku_loan/simulation/)
- 月額返済額・借入可能額・借り換え試算

#### [住信SBIネット銀行](https://www.netbk.co.jp/contents/lineup/home-loan/simulation/)
- ペアローン・収入合算の入力対応（合算後の返済額試算のみ）

#### [SBI新生銀行](https://www.sbishinseibank.co.jp/retail/housing/simulation/)
- 新規借入・借り換え・繰り上げ返済シミュレーション
- ペアローン・収入合算の年収合算入力に対応

#### [りそな銀行](https://www.resonabank.co.jp/kojin/jutaku/simulation/shinki/)
- 最大3件のシミュレーション結果を保存・比較可能
- ペアローン限定の団信プラン選択に対応

#### [auじぶん銀行](https://www.jibunbank.co.jp/products/homeloan/simulator/)
- 月額返済額の基本試算

### 比較・情報サイト系（第三者・中立）
#### [ルフトツールズ](https://housingloan.luft.co.jp/)
- 金利変動タイミングを複数指定可能
- 繰り上げ返済・住宅ローン減税・ボーナス払いに対応
- 返済予定表のダウンロード機能あり
- 未払利息の明示的な可視化・ペアローン対応は不明

#### [ゼロシステムズ](https://0systems.com/OneMinuteCheck/prepayment/)
- 5年ルール・125%ルールをオプションで適用可能
- 期間短縮型／返済額軽減型の繰り上げ返済比較
- 金利変動・最終回清算まで対応
- 月次グラフ・未払利息の推移表示・ペアローン対応はなし

#### [イー・ローン](https://www.eloan.co.jp/home/sim/)
- 金利変動シナリオを「上昇／一定／下降」で細かく指定可能（いつから・何回・どのくらい）
- 複数ローンの比較・グラフ表示あり
- 5年ルール・125%ルール・ペアローン独立計算への対応はなし

#### [モゲチェック](https://mogecheck.jp/simulation)
- 月額返済額・借り換えメリットの試算
- 金利上昇シナリオの詳細設定は薄い

#### [JLsim](https://jlsim.jp/)
- 住宅ローン返済と資産運用（インデックス投資）を組み合わせたシミュレーション
- 繰り上げ返済・金利変動の月次指定に対応
- 5年ルール・125%ルール・ペアローン対応は不明

#### [住宅金融支援機構（フラット35）](https://www.jhf.go.jp/simulation_loan/)
- 最大3プランを同時に試算・比較できる「返済プラン比較シミュレーション」
- 公的機関による中立ツール
- 変動金利の詳細シナリオ・ペアローン独立計算には非対応
